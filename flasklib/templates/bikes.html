How does PicABike work?


Many bikes are unlabeled or mislabeled on Craigslist, making it difficult to search for a bike. A simple text search for "road bike" will likely return a posting for "road bike shoes" but may not return a posting with a detailed description of an actual road bike. In regions where biking is popular, such as Seattle, over 3,000 bikes are available on Craigslist. Even if the user only spends half a second on each posting, that is still 25 minutes of lost time. To solve this problem, I developed PicABike, which uses the images in the postings to identify the bike type, making it easier for the user to "pic" the perfect bike.
