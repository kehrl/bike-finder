Why PicABike?

Many bikes are unlabeled or mislabeled on Craigslist. A simple text search for "road bike" will likely return a posting for "road bike shoes" but may not return a posting with a detailed description of an actual road bike. As a result, most users look through all postings to find a bike. In regions where biking is popular, such as Seattle, that can mean looking through over 3,000 bike postings; even if the user only spends half a second on each posting, that is still 25 minutes of lost time. To solve this problem, I developed PicABike, which uses the images in the postings to identify the bike type, making it easier for the user to "pic" the perfect bike.

How does PicABike work?

PicABike uses a convolutional neural network trained on 15,000 images of bikes from Craigslist, Ebay, and PinkBike to identify bike type from the posted image. New bike postings are scraped from Craigslist every three hours, labeled by bike type using the above algorithm, and added to the PicABike website.

More information is available at PicABike's Github page.
